Extract Constants

Do not hesitate to extract mappings of values like statuses into either enum's or constant object (deep immutability is supported through as const). Even if the constants are used in a single file this:


export  enum  AssessmentStatus {
  Ongoing,
  Completed,
  Archived,
}

Followed by this:

condition.status = AssessmentStatus.Completed;

Is more readable and maintainable then just:

condition.status = 2;

Comments are useful, however they become stale much more easily than code.

Use A Spell Checker

Using a spell checker is a great first line of defense against typos like carear instead of career or Questionaire instead of Questionnaire

Furthermore, although such typos are not intrinsically harmful, it is possible for such a typo to creep up in a network request which does have negative consequences. For example entering assesssment instead of assessment.
